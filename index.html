Get Started

Add this repo to F-Droid

https://northstar-works.github.io/fdroid/repo
1 Install F-Droid from f-droid.org if you haven't already
2 Open F-Droid → Settings → Repositories → tap +
3 Paste the repository URL above and tap OK
4 Refresh the repository list — Northstar Labs apps will appear
Scan to add on your device Point your Android camera or F-Droid's
built-in scanner at this code.
